# 🎨 Tech Icons Library (396 ícones)

Uma biblioteca centralizada com mais de 396 ícones de tecnologias, linguagens de programação e ferramentas de desenvolvimento, padronizados para uso em READMEs de perfil e documentação de projetos.

### 🚀 Como utilizar os ícones

Para usar qualquer ícone em seu projeto, utilize o link abaixo substituindo `NOME_DO_ARQUIVO` pelo nome da tecnologia que você precisa (ex: `csharp.png`, `react_native.png`, `3d.png`):

`https://raw.githubusercontent.com/mauriciospark/icones/main/NOME_DO_ARQUIVO.png`
